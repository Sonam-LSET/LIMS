# LIMS — Zoo Veterinary Lab Module (Mobile Prototype)

Interactive, self-contained HTML prototype of the **Laboratory Information Management System (LIMS)** mobile app for vets/paravets, part of a multi-site Zoo Management Platform.

## View it
- Open `index.html` in any browser (no build, no dependencies, works offline), **or**
- Enable **GitHub Pages** (Settings → Pages → Branch: `main` / root) to get a live URL.

## What's inside
Single-file prototype with a 5-tab bottom nav:

- **Requests** — Animal / Non-Animal lab requests, list + detail (navy multi-entity / white single-animal headers), creation flow, test & request cancellation.
- **Samples** — collection → confirm containers → barcode scan → ready to deliver → ship / pickup → in-transit, plus re-submit.
- **Tests** — tracker + catalogue.
- **Storage** — site & central stored samples, chain of custody, storage requests.
- **Reports** — *Veterinary Intelligence* module: Focus · Explore · Compare · Library, with Species / Animal / Site intelligence drill-downs.

> Prototype for design review. Sample data is illustrative.
